# Onshape Scriptable CAD Baseline

This folder holds the first scriptable CAD baseline for the single-tendon index-finger hardware concept.

## Why Onshape

Onshape supports two useful automation paths:

- `FeatureScript` for native parametric feature creation inside a Part Studio
- REST API access for external automation around documents, features, exports, and metadata

For this thesis-stage prototype, `FeatureScript` is the right first step because the geometry itself is the experiment.

## Files

- [single_tendon_ring_thimble.fs](/Users/rami/Documents/DTU/Thesis/soft_finger_thesis/cad/onshape/single_tendon_ring_thimble.fs): custom-feature body for a two-ring plus distal-thimble hardware set

## What The Feature Builds

The current feature creates three parts aligned along the finger axis:

- a proximal guide ring
- a middle guide ring
- a distal thimble with a closed tip cap

Each part includes:

- a finger bore
- a longitudinal tendon channel offset to one side
- a simple split cut so the ring or thimble can be opened

The thimble also includes:

- a transverse distal anchor hole for the tendon

## Coordinate Convention

- `+X`: distal direction along the finger
- `+Y`: tendon-guide side
- `+Z`: dorsal direction

## How To Use In Onshape

1. Create a new document and add a `Feature Studio`.
2. Keep the two autogenerated header lines at the top of the studio.
3. Paste the contents of [single_tendon_ring_thimble.fs](/Users/rami/Documents/DTU/Thesis/soft_finger_thesis/cad/onshape/single_tendon_ring_thimble.fs) below that header.
4. Commit the Feature Studio.
5. In a `Part Studio`, use `Add Custom Features` and add the new feature to the toolbar.
6. Insert the feature and start with a simple index-finger-sized parameter set.

## Suggested First Parameters

These are only starting values, not validated anatomy:

- proximal ring center X: `0 mm`
- middle ring center X: `22 mm`
- guide ring inner diameter: `19 mm`
- guide ring length: `8 mm`
- guide ring wall: `2 mm`
- guide ring split width: `4 mm`
- tendon channel diameter: `2 mm`
- tendon guide offset Y: `13 mm`
- tendon guide offset Z: `3 mm`
- guide boss wall: `1.5 mm`
- thimble start X: `36 mm`
- thimble inner diameter: `16 mm`
- thimble length: `22 mm`
- thimble wall: `2 mm`
- thimble tip cap: `4 mm`
- thimble split width: `5 mm`
- anchor hole diameter: `2 mm`

## Current Limitations

- no ergonomic contouring
- no tendon wrapping, friction, or pulley geometry
- no mating features between the soft body and a textile substrate
- no explicit printability checks
- no linkage yet to the reduced-order Python routing model

## Immediate Next CAD Iterations

- make ring locations driven by phalanx landmarks instead of raw X coordinates
- split the thimble into a softer sleeve and a stiffer distal anchor insert
- add a second routing family with a more dorsal return path
- export consistent geometry measurements back into the simulation-modeling stack
