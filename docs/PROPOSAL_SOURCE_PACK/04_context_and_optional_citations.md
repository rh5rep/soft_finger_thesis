## Practical Rule

If a paper does not help answer one of these proposal questions, keep it out of the main text:

1. Why this movement problem matters
2. Why this task framing is defensible
3. What direct hardware precedents exist
4. Why the chosen model and validation level are appropriate

## Likely Overuse Risks

- `Zhang 2023` can pull the proposal too far toward gait cueing.
- `Rose 2019` can pull the proposal too far toward auditory cueing rather than finger-device mechanics.
- `van der Kooij 2022` and `Roda-Casanova 2024` can make the proposal look pinch-centered if cited too heavily.
- `Raciti` papers can make the proposal sound like a clinical rehabilitation trial rather than a simulation-plus-benchtop thesis.
